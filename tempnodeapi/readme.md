# tempnodeApi

Para inicializar el proyecto:

```shell
npm install
````
## Arranque
Para arrancar el proyecto usar:
* En producción:
```shell
npm start
```

* En desarrollo en el puerto localhost:3000:
```shell
npm run dev
```

## Rutas del API
* http://localhost:3000/apiv1/agentes : Retorna una lista de agentes
